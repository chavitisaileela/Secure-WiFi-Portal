# ShieldGate WiFi Portal

## Installation
1. Download XAMPP from: https://www.apachefriends.org/
2. Put these files in `C:\xampp\htdocs\wifi_project`.
3. Open XAMPP and start Apache and MySQL.

## Important Links
* **Local Website:** http://localhost/wifi_project/index.html
* **Admin Database:** http://localhost/phpmyadmin/
